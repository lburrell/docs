PHEMI provides a graphical user interface to configure and use a data source. The following screenshot shows a list of data sources:

When a data source is selected, a screen is displayed showing:

  • Data Policy - Provide the information such as ownership, privacy, data sharing and other policy information
  • Ingest Data - Provides a method of ingesting data by uploading a single or set of files
  • Data Processing Function - The DPF to be used to process the ingested data to generate a set of derived data
